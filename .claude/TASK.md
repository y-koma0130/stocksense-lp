# 概要
このリポジトリは**StockSense**という日本の個別株を中心として銘柄の割安株を通知するサービスのLPです。
このLPでは
- サービスの簡単な説明
- 利用ユースケース
- LINE連携用のQRコードの動線
- 利用規約
- プライバシーポリシー
などの簡易的なページをAstroフレームワークを使って構築します。
作成後はcloudflare pagesにデプロイします。
サービス自体のリポジトリは `../stocksense` にあります。

# 注意事項
- ユーザーに確認が必要な場合、判断が必要な場合は勝手に進めずに聴くこと
- 載せる情報は正確に伝えられるように心がけ、嘘は絶対に載せないこと
- 必要な素材等があればユーザーに求めること
- 言語は日本語で出力すること
